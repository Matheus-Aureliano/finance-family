# 💰 FinanceFamily - Controle Financeiro Familiar

O **FinanceFamily** é uma aplicação web simples e intuitiva para ajudar famílias a organizarem suas finanças. Com ele, é possível cadastrar ganhos e gastos, visualizar um extrato detalhado e manter o histórico salvo diretamente no navegador.

---

## 🚀 Funcionalidades

* **Sistema de Cadastro e Login:** Proteção simples utilizando validação de e-mail e armazenamento local.
* **Lançamentos:** Adicione descrições, valores e categorias (Alimentação, Lazer, Contas, Receitas).
* **Extrato Dinâmico:** Visualização automática de itens com cores diferenciadas para entradas (verde) e saídas (vermelho).
* **Persistência de Dados:** Os dados não se perdem ao fechar a página (utiliza `localStorage`).
* **Responsividade:** Interface limpa que se adapta a diferentes tamanhos de tela.

---

## 🛠️ Tecnologias Utilizadas

Este projeto foi construído "puro" (Vanilla), sem frameworks externos, para demonstrar domínio das bases da web:

* **HTML5:** Estrutura e semântica.
* **CSS3:** Estilização e layout.
* **JavaScript (ES6+):** Lógica de validação, manipulação do DOM e persistência em memória local.

## 🔧 Como rodar o projeto

Como o projeto é estático, você não precisa instalar nada!

1. Faça o download ou clone este repositório.
2. Abra o arquivo `index.html` no seu navegador de preferência.
3. Ou acesse o link oficial: [Acesse o FinanceFamily aqui](https://matheus-aureliano.github.io/finance-family/)

---

## 📝 Notas técnicas

Este projeto utiliza o **LocalStorage** do navegador para salvar as informações. Isso significa que:
1. Os dados são salvos apenas no dispositivo onde foram digitados.
2. Se você limpar o cache do navegador, os dados serão apagados.
3. É uma excelente solução para ferramentas de uso pessoal e demonstração de lógica de programação.
